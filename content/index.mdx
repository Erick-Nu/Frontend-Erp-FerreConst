# ESNT Ferreteria API Documentation

> Documentación completa de la API REST para el sistema de gestión de ferretería.

Bienvenido a la documentación oficial de la API de **ESNT Ferreteria**. Esta API permite gestionar empresas, usuarios, productos, stock, proformas y alertas de stock bajo.

## Stack Técnico

- **Runtime**: Node.js >= 22.22.1
- **Framework**: Express 5 + TypeScript
- **Base de datos**: PostgreSQL
- **Autenticación**: JWT (access + refresh tokens)
- **Documentación**: OpenAPI 3.1

## Base URL

```
http://localhost:3000
```

## Ejemplo Rápido

### Login

```bash
curl -X POST http://localhost:3000/auth/login \
  -H "Content-Type: application/json" \
  -d '{
    "emruc": "1709639106001",
    "usapodo": "jperez",
    "uspassword": "secreto123"
  }'
```

### Obtener Productos

```bash
curl http://localhost:3000/products \
  -H "Authorization: Bearer <tu_access_token>"
```

## Estructura de la API

La API sigue una arquitectura REST con los siguientes módulos principales:

| Módulo | Descripción | Endpoints |
|--------|-------------|-----------|
| [Auth](/api/auth/login) | Autenticación y sesiones | 3 |
| [Companies](/api/company/register-company) | Gestión de empresas | 5 |
| [Users](/api/user/register-user) | Gestión de usuarios | 6 |
| [Products](/api/product/register-product) | Catálogo de productos | 4 |
| [Stock](/api/stock/register-stock) | Control de inventario | 5 |
| [Proformas](/api/proforma/register-proforma) | Generación de proformas | 7 |
| [Alerts](/api/alert/get-alerts) | Alertas de stock bajo | 3 |
